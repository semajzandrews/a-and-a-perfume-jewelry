# REFERENCE — external award-grade study (perfume / jewelry / gifting)

Studied fresh for this build (concept-level only; no internal build referenced).

1. **Aesop / Le Labo / Maison Margiela "Replica" apothecary web direction** —
   restrained warm-neutral grounds, label-led typography, ingredient lists treated
   as editorial copy, generous whitespace. Takeaway: let the *composition* and the
   ingredient language be the luxury, not gradients or chrome. Drives the
   parchment ground + Boska "label" voice + the scent-note copy.

2. **High-jewelry editorial (Awwwards: Boucheron / Vanrycke-class sites)** —
   alternating full-bleed image bands with quiet typographic interludes, macro
   detail photography, brass/hairline rules. Takeaway: alternating horizontal
   bands carry a gift shop beautifully when each band has one clear job. Drives the
   stacked alternating-band layout (lane) and brass hairlines.

3. **Niche-fragrance "scent pyramid" interactives** — perfumers visualize a scent
   as top/heart/base strata. Rarely done well on a small-shop site. Takeaway: an
   interactive olfactory pyramid is the brand-true signature that pairs scent with
   adornment. Drives the signature moment.

4. **Artisan gifting / boutique lookbooks (SiteInspire warm-editorial)** —
   numbered chaptering, "the counter / the case / the gift" narrative, an
   in-store, walk-in-and-be-helped tone. Takeaway: sell the *experience of being
   helped to choose a gift*, not a product catalog. Drives section structure and
   the warm, personal voice.

## Art-direction decisions
- Ground: warm parchment `#F2EBDD` / `#EAE0CC`; ink cacao `#3A2A1E`; accent
  `#6F4E37`; brass hairline `#B08A52`; rose-quartz whisper `#C9A6A0` (sparing).
- Type: Boska (display, the labels + headlines) + Satoshi (body) + a mono caption
  for note-counts. No pairing reused from registry.
- Imagery: macro perfume bottles + amber liquid, gold/jewelry detail, gift
  wrapping, hands presenting a box. Each image used ONCE.
- Motion: GSAP ScrollTrigger for the pyramid build + band reveals; Lenis smooth;
  restraint elsewhere. No scroll-snap, no autoplay-video pileup.
